# 4. Objetivos

## 4.1. General

Implementar un sistema para la biblioteca Constelación de Libros, con el fin de gestionar de manera eficiente el inventario de libros y productos, el registro de clientes, el control de préstamos, ventas, proveedores y facturación, garantizando información organizada, confiable y accesible para mejorar la administración y la atención al usuario.

## 4.2. Específicos

- Administrar de forma digital la información de clientes, empleados, proveedores, libros y productos de papelería.
- Optimizar los procesos de préstamos y devoluciones de libros, evitando pérdidas de información y mejorando el seguimiento a los usuarios.
- Facilitar el control de ventas y facturación, garantizando precisión en los cálculos y reportes generados.
